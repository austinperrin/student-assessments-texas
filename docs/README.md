# Reference PDFs

This folder stores the local TEA and Texas Assessments PDF reference files used to build the mapping JSON files in this repo.

PDFs are organized into year-based subfolders to mirror the way the source documents are grouped online.

Use the ending year when a PDF is labeled by school year.

Examples:

- `docs/2024/2023-2024-staar-interim-data-file-format.pdf`
- `docs/2025/2024-2025-crs-data-file-format.pdf`
- `docs/2026/2025-2026-file-naming-convention.pdf`

For single-year PDFs, use that same year as the folder name.

Examples:

- `docs/2018/2018-staar-eoc-data-file-format.pdf`
- `docs/2024/2024-tfar-data-file.pdf`

This folder is local reference storage only.
The source of truth for each mapping file's official document link should remain the `metadata.pdf_url` value stored in the JSON.
