# AP Source Archive

This folder stores AP source documents organized by delivered score-report
family.

## Structure

- `k12_score_reports/`
- `higher_ed_score_reports/`

Store year-specific source documents beneath the matching score-report family.

## Archived Sources

- `k12_score_reports/2026/`, `2025/`, and `2024/` contain AP student datafile
  layout PDFs for district and school reporting
- `higher_ed_score_reports/2026/` and `2025/` contain higher-ed student data
  record layouts for both TXT and CSV deliveries

Use these archived PDFs as the source of truth for year-specific AP mapping
differences before normalizing any field names or assumptions across years.
