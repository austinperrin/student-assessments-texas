# SAT Suite Source Archive

This folder stores SAT Suite source documents organized by delivery family and
assessment variant.

## Structure

- `k12_reporting/`
- `higher_ed_reporting/`

Store year-specific source documents beneath the matching delivery family and
assessment child folder.

## Archived Sources

- `k12_reporting/sat/2026/` contains the `2025-26` SAT electronic score report
  layout
- `k12_reporting/sat/2025/` contains the `2024-25` SAT electronic score report
  layout
- `k12_reporting/psat_nmsqt/2026/` and `2025/` contain the combined
  PSAT/NMSQT and PSAT 10 electronic score report layouts
- `k12_reporting/psat_8_9/2026/` and `2025/` contain the PSAT 8/9 electronic
  score report layouts
- `higher_ed_reporting/sat/2026/` and `2025/` contain the SAT higher-ed
  electronic score report layouts

## Known Gaps

- the older `2023-24` SAT Suite PDFs were discovered in search indexing but
  were not downloadable from the current public path during this pass
- `psat_10/` remains a logical scaffold folder, but the official K-12 layout
  document is shared with `psat_nmsqt/`
