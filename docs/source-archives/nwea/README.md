# NWEA Source Archive

This folder stores local NWEA source reference material used to curate and
validate mapping files.

## Purpose

Add NWEA product pages, help-center references, spreadsheets, and archived
layout documents here as they are downloaded and validated.

## Usage

- create assessment-family and delivery-format directories as needed beneath
  this folder
- keep official source documents aligned to current NWEA product or help-center
  documentation
- reference these files using repo-relative links from NWEA mapping metadata
  and `README.md` documentation

## Structure

- `map_growth/`

Use assessment-family folders first, then add delivery-format and year folders
only where the source set benefits from that extra distinction.
