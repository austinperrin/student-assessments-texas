# Pre-AP Agent Guide

This file guides AI agents working inside `assessments/collegeboard/pre_ap/`.

## Scope

Use this folder for Pre-AP product mappings and related maintenance docs.

## Working Rules

- keep Pre-AP mapping assets under `assessments/collegeboard/pre_ap/`
- preserve official Pre-AP source-document metadata and archive references
- keep classroom-assessment outputs separate from other College Board product
  families
- update `docs/source-archives/collegeboard/pre_ap/` with Pre-AP source files
  before publishing mapping changes

## Agent Expectations

- create year- or product-specific mapping files only after the Pre-AP format is
  confirmed
- choose the classroom-assessment delivery family before creating year-specific
  mappings
- do not assume a single culminating exam layout because Pre-AP is a classroom
  assessment program rather than a one-test exam family
- keep field names aligned with College Board source semantics
- preserve source ordering, omitted blanks, and header uniqueness in mappings
