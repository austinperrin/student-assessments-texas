# Pre-AP

This folder stores College Board Pre-AP mapping scaffolding and maintenance
guidance.

## Purpose

Use this family for Pre-AP result files and related artifacts. Current
scaffolding places them under the classroom assessment delivery family:

- `classroom_assessments/`

## Current Status

This folder is a scaffold. Add mapping files once the Pre-AP source formats are
confirmed and archived in the matching delivery-family archive under
`docs/source-archives/collegeboard/pre_ap/`.

## Working Notes

- Keep Pre-AP separate from AP and the SAT Suite.
- Pre-AP does not have a final exam; when mapping here, expect classroom and
  program assessment artifacts rather than a single annual test layout.
