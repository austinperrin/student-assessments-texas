# SAT Suite Higher-Ed Reporting

This folder stores SAT Suite mapping scaffolding for higher-education reporting
files delivered to colleges and universities.

## Purpose

Use this delivery family for higher-ed SAT Suite result files before splitting
into assessment-specific child folders.

Current scaffolded assessment variants include:

- `sat/`

## Current Status

This folder is a scaffold. Add mapping files once the relevant higher-ed SAT
Suite source formats are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/higher_ed_reporting/`.

## Working Notes

- Prefer this folder when the source material is aimed at colleges,
  universities, or higher-ed reporting workflows.
- Keep this delivery family separate from K-12 reporting even when the
  underlying assessment brand is the same.
