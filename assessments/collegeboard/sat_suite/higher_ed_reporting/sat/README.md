# SAT Higher-Ed Reporting

This folder stores College Board SAT higher-ed reporting mapping files and
maintenance guidance.

## Purpose

Add SAT higher-ed reporting mappings, coverage notes, and source-document
references here.

## Current Status

This folder is a scaffold. Add mapping files once the SAT higher-ed reporting
source formats are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/higher_ed_reporting/sat/`.

## Working Notes

- Use this folder only for higher-ed SAT reporting outputs.
- Keep K-12 SAT outputs under `sat_suite/k12_reporting/sat/`.
