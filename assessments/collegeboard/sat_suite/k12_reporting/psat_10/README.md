# PSAT 10

This folder stores College Board PSAT 10 mapping files and maintenance
guidance.

## Purpose

Add PSAT 10 product mappings, coverage notes, and source-document references
here.

## Current Status

This folder is a scaffold. Add mapping files once the PSAT 10 source formats
are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/k12_reporting/psat_10/`.

## Working Notes

- Keep PSAT 10 under the SAT Suite family rather than under a standalone
  `psat/` family.
- College Board documents PSAT 10 and PSAT/NMSQT as the same test delivered in
  different administrations; map them separately only when the source layout or
  reporting output differs.
- Use this folder only for K-12 reporting outputs.
