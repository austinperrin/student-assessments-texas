# PSAT/NMSQT

This folder stores College Board PSAT/NMSQT mapping files and maintenance
guidance.

## Purpose

Add PSAT/NMSQT product mappings, coverage notes, and source-document references
here.

## Current Status

This folder is a scaffold. Add mapping files once the PSAT/NMSQT source formats
are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/k12_reporting/psat_nmsqt/`.

## Working Notes

- Keep PSAT/NMSQT under the SAT Suite family rather than under a standalone
  `psat/` family.
- College Board documents PSAT/NMSQT and PSAT 10 as the same test delivered in
  different administrations; map them separately only when the source layout or
  reporting output differs.
- Use this folder only for K-12 reporting outputs.
