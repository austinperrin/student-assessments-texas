# PSAT 8/9

This folder stores College Board PSAT 8/9 mapping files and maintenance
guidance.

## Purpose

Add PSAT 8/9 product mappings, coverage notes, and source-document references
here.

## Current Status

This folder is a scaffold. Add mapping files once the PSAT 8/9 source formats
are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/k12_reporting/psat_8_9/`.

## Working Notes

- Keep PSAT 8/9 under the SAT Suite family rather than under a standalone
  `psat/` family.
- Treat PSAT 8/9 as its own assessment within the suite rather than as a
  simple administration variant of PSAT/NMSQT or PSAT 10.
- Use this folder only for K-12 reporting outputs.
