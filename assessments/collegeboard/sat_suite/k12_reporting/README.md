# SAT Suite K-12 Reporting

This folder stores SAT Suite mapping scaffolding for K-12 reporting files
delivered to districts, schools, or K-12 educators.

## Purpose

Use this delivery family for K-12 SAT Suite result files before splitting into
assessment-specific child folders.

Current scaffolded assessment variants include:

- `sat/`
- `psat_nmsqt/`
- `psat_10/`
- `psat_8_9/`

## Current Status

This folder is a scaffold. Add mapping files once the relevant K-12 SAT Suite
source formats are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/k12_reporting/`.

## Working Notes

- Prefer this folder when the source material is aimed at school, district, or
  K-12 educator reporting workflows.
- Keep PSAT/NMSQT and PSAT 10 separate only when the delivered layouts differ.
