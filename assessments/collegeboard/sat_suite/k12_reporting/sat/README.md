# SAT K-12 Reporting

This folder stores College Board SAT K-12 reporting mapping files and
maintenance guidance.

## Purpose

Add SAT K-12 reporting mappings, coverage notes, and source-document references
here.

## Current Status

This folder is a scaffold. Add mapping files once the SAT K-12 reporting source
formats are confirmed and archived in
`docs/source-archives/collegeboard/sat_suite/k12_reporting/sat/`.

## Working Notes

- Use this folder only for K-12 SAT reporting outputs.
- Keep higher-ed SAT outputs under `sat_suite/higher_ed_reporting/sat/`.
