# AP K-12 Score Reports

This folder stores AP K-12 score-report mapping scaffolding and maintenance
guidance.

## Purpose

Use this delivery family for AP score-report files delivered to schools,
districts, or K-12 educator workflows.

## Current Status

This folder is a scaffold. Add mapping files once the AP K-12 score-report
source formats are confirmed and archived in
`docs/source-archives/collegeboard/ap/k12_score_reports/`.
