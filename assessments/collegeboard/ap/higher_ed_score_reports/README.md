# AP Higher-Ed Score Reports

This folder stores AP higher-ed score-report mapping scaffolding and
maintenance guidance.

## Purpose

Use this delivery family for AP score-report files delivered to colleges,
universities, or higher-ed workflows.

## Current Status

This folder is a scaffold. Add mapping files once the AP higher-ed score-report
source formats are confirmed and archived in
`docs/source-archives/collegeboard/ap/higher_ed_score_reports/`.
