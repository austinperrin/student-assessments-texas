# AP

This folder stores AP mapping scaffolding and maintenance guidance.

## Purpose

Use this family for AP result files, organized first by the audience-specific
score-report family that delivers the data.

Current scaffolded delivery families include:

- `k12_score_reports/`
- `higher_ed_score_reports/`

## Current Status

This folder is a scaffold. Add mapping files once the AP source formats are
confirmed and archived in the matching delivery-family archive under
`docs/source-archives/collegeboard/ap/`.

## Working Notes

- Keep AP separate from the SAT Suite family.
- Expect both K-12 educator score-report layouts and higher-ed score-report
  layouts in current and historic AP documentation; do not collapse them into a
  single assumed output format without source confirmation.
