This file guides AI agents working inside
`assessments/collegeboard/accuplacer/tsia2/`.

## Scope

Use this folder for TSIA2 product mappings and related maintenance docs.

## Working Rules

- keep TSIA2 mapping assets under `assessments/collegeboard/accuplacer/tsia2/`
- preserve official TSIA2 source-document metadata and archive references
- keep TSIA2 nested under ACCUPLACER when College Board is the governing source
- update `docs/source-archives/collegeboard/accuplacer/tsia2/` with TSIA2
  source files before publishing mapping changes

  confirmed
