# ACCUPLACER Platform Reports

This folder stores ACCUPLACER platform-report mapping scaffolding and
maintenance guidance.

## Purpose

Use this delivery family for generic ACCUPLACER platform reports, score
exports, and related delivered file layouts that are not specific to TSIA2.

## Current Status

This folder is a scaffold. Add mapping files once the ACCUPLACER platform
report source formats are confirmed and archived in
`docs/source-archives/collegeboard/accuplacer/platform_reports/`.
