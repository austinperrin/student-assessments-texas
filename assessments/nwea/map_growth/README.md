# MAP Growth

This folder groups MAP Growth mapping assets and delivery-format guidance.

## Scope

Current NWEA help content describes scheduled MAP Growth result exports in CSV
format, including combined and comprehensive export packages, with optional
student-class, program, and accommodation data in the comprehensive package.

## Current Delivery Families

- `data_exports/`
  Home for machine-readable MAP Growth export layouts such as combined and
  comprehensive CSV exports.

Operational or diagnostic reports that are primarily human-facing should only
be added here if the project begins maintaining mappings for those report
layouts.
