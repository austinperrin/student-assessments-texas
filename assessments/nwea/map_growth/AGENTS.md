# MAP Growth AI Agent Guide

This file guides AI agents working inside `assessments/nwea/map_growth/`.

## Scope

Use this area for MAP Growth mapping assets and family-level maintenance docs.

## Working Rules

- keep MAP Growth assets under `assessments/nwea/map_growth/`
- add durable delivered-file families such as `data_exports/` before adding
  year-specific mappings
- treat MAP Spanish support as part of MAP Growth when the source document
  describes the same delivered export or report format
- update `docs/source-archives/nwea/map_growth/` with the matching source files
  before or alongside mappings when local archival copies are needed
- keep MAP Growth field names and file families aligned to MAP Growth sources

## Mapping Expectations

- use year- or effective-period-specific files once official field references
  are archived or otherwise confirmed
- preserve the export family described by NWEA, such as combined versus
  comprehensive export data, when those outputs differ materially
- keep official source URLs aligned to the current NWEA help or product pages
